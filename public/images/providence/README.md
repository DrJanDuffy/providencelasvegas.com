# Providence Images & Assets

Optimized folder structure for Providence Las Vegas images, photos, logos, and maps.

## Folders

| Folder | Purpose |
|--------|---------|
| `community-resources/` | HOA card graphics, icons (e.g. New Homeowner, Pay Assessment, Community Parks, About Providence) |
| `parks/` | The Promenade, Knickerbocker Park, Huckleberry Park photos |
| `logos/` | Providence / HOA logos (use only if you have rights) |
| `maps/` | Providence community map |
| `neighborhoods/` | Optional: per-neighborhood photos |
